# INFS 740 Group Project

## Installation

* Download nodejs 8.4.0
* Download npm
* Run `npm install`

## Starting the App

* Create a `.env` file with the content `PRIAID_API_KEY=email` and `PRIAID_SECRET_KEY=secret` (replace tjese with the API keys obtained from apimedic.com/apitest)
* Run `npm run start`
